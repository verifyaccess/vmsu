Service rules

General rules

  • By registering on the site you automatically agree with service rules
  • There’s no minimum payment or registration fee on the site
  • Funds are credited automatically
  • The funds credited to the user account cannot be refunded
  • Administration reserves the right to refuse rendering services, registration or delete an account without explanation

Invitation system

  • The service has a system of invitations for new users
  • Each user can invite as many partners as he or she wants
  • A new invitation is generated for each new user. The invitation code is a one-time code that can be used for registration just once
  • The user bears responsibility for each invited user with his/her account and account balance
  • Posting invitation codes to any message boards is forbidden

Using checker

  • The buyer uses Checker at his/her own risk

  • The buyer pays for Checker regardless of the status
